# Base Site

Main repo: [Gitlab](https://gitlab.jmj30yt.xyz/jmj30/)

Uses [UV](https://github.com/astral-sh/uv)

## How to use

Open terminal and run (one of them):

UV (Default way):
`uv run main.py`

Reg Python:
`python -m venv .venv`
`python -m pip install interactions-py dotenv`

Docker:
`docker build -t Site .`

Docker compose:
`docker compose up`
